# ygst_sob_login_demo
不知道有没有小伙伴，跟我一样，刚刚接触阳光沙滩就卡在了接入登陆API上的，这个项目主要是一个很简单的登录api代码。

阳光沙滩的登录demo，没有其他的框架，代码是Java用到okHttp发送请求，使用glide加载验证码的图片。

登陆主要是要在post的时候用到的三个参数，账号、密码、验证码，还有一个就是请求头参数l_c_i
